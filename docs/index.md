## Overview

SkillForge is an internal training platform where employees complete assigned courses and admins manage content, users, invitations, deadlines, and progress reporting.

This documentation is written for both developers and product owners. It is organized so you can get the app running quickly, understand system boundaries, and work safely with API and content tooling.

## What Is Covered

- Local setup for backend, frontend, and docs
- Role model and route access rules
- Employee workflow from dashboard to quiz and results
- Admin workflow for content, users, invites, and assignments
- API endpoint reference grouped by domain
- Markdown authoring features used by lessons
