# Employee Guide

This guide explains how employees use SkillForge.

---

## Dashboard

After login, employees land on the dashboard.

It shows:

- total courses
- completed courses
- in-progress courses
- average quiz score

Courses are grouped into:

- Continue Learning
- Not Started
- Completed

---

## Courses

Employees can:

- browse all published courses
- open a course to start learning

Each course shows:

- progress %
- number of lessons
- number of quizzes
- average score

---

## Course View

Inside a course:

- sidebar → all lessons and quizzes
- main area → current content

---

## Lessons

Lessons are written in Markdown.

To complete a lesson:

- click **Next**
- or **Complete Lesson**

---

## Quizzes

Quizzes are part of the course flow.

When starting a quiz:

- answer all questions
- submit answers
- receive score instantly

---

## Passing a Quiz

To pass:

```
Score ≥ 80%
```

If failed:

- attempt is saved
- does NOT count toward completion
- can retry

---

## Progress Tracking

Progress is based on:

- completed lessons
- passed quizzes

---

## Results Page

Shows:

- overall stats
- per-course breakdown
- recent quiz attempts
- best scores

---

## Reset Progress

If enabled, employees can:

- reset all progress
- restart courses from scratch

---

## Restrictions

Employees cannot:

- edit content
- access admin pages
- see draft content
