# Посібник для співробітника

Цей посібник описує повний досвід співробітника в SkillForge.

## Доступ і навігація

Після входу співробітника перенаправляють на `/` (dashboard).

Доступні сторінки співробітника:

- Dashboard (`/`)
- Courses (`/courses`)
- Course view (`/courses/:id`)
- Results (`/results`)

Співробітникам заборонено доступ до всіх сторінок `/admin/*`.

## Dashboard

Dashboard показує:

- кількість призначених курсів за статусами
- інсайти щодо середнього балу
- згруповані секції курсів, наприклад continue learning і completed
- контекст призначень, зокрема дедлайни та статус термінів

Використовуйте цю сторінку як стартову для щоденних навчальних задач.

## Список курсів

Сторінка курсів показує опубліковані курси, доступні співробітнику, з метаданими по кожному курсу, включно з:

- відсотком прогресу
- кількістю уроків і квізів
- середнім балом
- сигналами щодо дедлайнів призначення

## Перегляд курсу

Курс поєднує уроки й квізи в один впорядкований потік.

Сторінка містить:

- sidebar із контентом (уроки та квізи)
- панель контенту для вибраного елемента
- контекстний статус завершення

## Завершення уроку

Уроки — це навчальні одиниці, побудовані на Markdown.

Урок вважається завершеним, коли співробітник активує завершення уроку через сценарій курсу.

## Завершення квізу

Поведінка квізу:

- відповіді відправляються і оцінюються одразу
- кожна спроба записується в історію
- поріг проходження становить 80%

Якщо квіз не пройдено, його можна спробувати ще раз. Невдалі спроби зберігаються, але не вважаються проходженням.

## Логіка завершення курсу

Курс вважається завершеним лише тоді, коли:

- усі уроки завершено
- усі квізи пройдено

## Сторінка результатів

Перегляд результатів агрегує особисту ефективність, зокрема:

- знімки прогресу на рівні курсу
- історію спроб квізів
- тренди балів і найкращі спроби

## Скидання прогресу

Співробітники можуть скинути власний прогрес зі сторінки результатів, щоб почати навчання з нуля.

## Правила видимості

Співробітники бачать лише опублікований контент.

Чернетки курсів, уроків і квізів залишаються прихованими, доки адміністратор їх не опублікує.
