## Огляд

SkillForge — це внутрішня платформа навчання, де співробітники проходять призначені курси, а адміністратори керують контентом, користувачами, запрошеннями, дедлайнами та звітністю по прогресу.

Ця документація призначена і для розробників, і для власників ПЗ. Вона побудована так, щоб ви могли швидко запустити застосунок, зрозуміти межі системи та безпечно працювати з API й інструментами контенту.

## Що охоплено

- локальне налаштування backend, frontend і docs
- модель ролей і правила доступу до маршрутів
- шлях співробітника від дашборда до квізу й результатів
- робочий процес адміністратора для контенту, користувачів, запрошень і призначень
- довідник API-ендпоінтів, згрупований за доменами
- можливості Markdown, які використовуються в уроках
